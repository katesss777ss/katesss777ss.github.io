Wuthering waves

Разработчик мобильной игры Punishing: Gray Raven компания Kuro Game выпустила дебютный геймплейный трейлер Wuthering Waves,
новой ролевой игры с открытым миром, находящейся в данный момент разработке

Компания Kuro Game опубликовала вступительный ролик для кроссплатформенной экшен-ролевой игры с открытым миром Wuthering Waves
в преддверии старта второго закрытого бета-тестирования 19 февраля.

Разработчики учли отзывы игроков с прошлых тестов и внесли некоторые коррективы в сюжет игры,
боевую систему, графическую составляющую и контент открытого мира. Команда проекта прекрасно осознает,
что в игре все еще много недостатков, и отзывы и предложения тестеров помогут им представить игру, которая понравится игрокам.

Разработчики трехмерного экшена с открытым миром для ПК, консолей и мобильных Wuthering Waves из
студии Kuro Game опубликовали новый пост в своем блоге с информацией о части изменений грядущего закрытого
бета-тестирования из проекта относительно предыдущих версий игры. Одним из, вероятно, наиболее приятных бонусов
для многих игроков, которые не любят читать кучи диалогов, будет кнопка Skip для пропуска таковы
